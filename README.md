# Event_manager

> A basic event management website

- Create users
- login and logout
- Create Events
- see events of different users
- invite others 

## Built With

- Rails 





## Getting Started

- `git clone https://github.com/iam-Akshat/event-manager.git`
- `cd event-manager`
- `bundle install`
- `rails db:migrate`
- `rails s`

### Prerequisites
 - Ruby
 - Rails
